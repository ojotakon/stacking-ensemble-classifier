# Manual Stacking Ensemble Classifier (scikit-learn)

This project demonstrates how to implement a stacking ensemble model manually 
without using `sklearn.ensemble.StackingClassifier`.

## Features
- Manual out-of-fold (OOF) prediction generation  
- Meta-learner training  
- Base model comparison  
- Voting ensemble comparison  
- Clean, reproducible pipeline  

## Technologies
- Python  
- scikit-learn  
- NumPy / Pandas
